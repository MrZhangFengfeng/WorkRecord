## 方法自执行
react的render方法内，如果return中的某个标签内写了方法


    render() {
            return (
                <div>
                    <span onClick={alert("winter")}/>
                    <span onClick={console.log("winter")}/>
                    <span onClick={!function(){...}()}/>
                    <span onClick={this.fn()}/>
                </div>
            )
        }
        
会在页面渲染的时候直接执行onClick对应的方法。

只有这样不带括号才不会自动执行：`<span onClick={this.fn}/>`
